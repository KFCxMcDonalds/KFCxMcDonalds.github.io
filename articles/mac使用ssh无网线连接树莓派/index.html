Mac使用ssh无网线连接树莓派

Mac使用ssh无网线连接树莓派

不使用鼠标键盘和网线通过ssh连接树莓派

前提

树莓派开机后自动连接好WiFi并开启ssh服务，具体文章见初见树莓派：系统安装或树莓派WiFi连接和开启ssh。

1 连接步骤

  • 搜索树莓派IP地址
  • ssh远程连接树莓派terminal

1.1 搜索树莓派IP地址

本文面向不使用键盘和屏幕连接树莓派的读者，所以需要使用电脑或路由器来查找同一网络下树莓派的IP。这里介绍常用的几种方式，更多可以查看官网doc

🦁路由器

浏览器打开路由器地址:192.168.1.1，登陆路由器账户并进入控制面板。

打开已连接设备列表，可以看到连接到本路由器的所有设备，通过排除能够认识的设备，比如电脑、打印机、手机等，找到树莓派的IP地址。

🐱使用支持mDNS的设备查找

如果你的设备支持mDNS，那么可以通过ping树莓派的主机名来查找IP

Info
主机名可以在安装系统时在Imager的Advance Option中设置，详见文章初见树莓派：系统安装，默认为raspberry.local）：

用网线连接电脑和树莓派，输入命令

1

ping raspberry.local

如果可以找到该主机名，那么ping命令将显示ping到的地址：

🐯nmap命令

安装

nmap(Network Mapper)是一个开源的网络查找工具。

  • Linux在命令行输入apt install nmap下载
  • macOS和Windows的下载请查看nmap官网
使用

要使用nmap查找树莓派的IP，首先需要知道你电脑的本机IP地址。

  • Linux在命令行输入hostname -I
  • macOS系统设置-网络-详细信息- IP地址
  • Windows控制面板-网络-显示网络连接-选择网络-显示连接信息

然后使用nmap查找本网段中的所有设备，比如你的本机IP为192.168.1.4，那么输入命令：nmap -sn 192.168.1.0/24

🐶软件搜索

手机可以使用软件Fing，查找本地网络中的树莓派。

macOS我使用的免费的LanScan，可以找到网络中的树莓派。

Note
如果按照教程完成了操作系统的安装和网络的配置还是找不到树莓派IP的话，可能是提供给树莓派的电源电压不足，必须是5V3A，如果电源功率不足黄灯可能规律的2次闪烁或者亮度很低。 正常启动的黄灯是不规律闪烁的，代表SD卡在写入读取数据。

1.2 ssh连接树莓派

打开terminal，输入ssh 用户名@ip地址，会要求输入密码，输入后即可连接成功。

  • 如果用Imager写入的系统，那么登陆时就使用当时设置的用户名和密码
  • 如果是老版本自己写入的镜像，那么默认用户名为pi，密码是raspberry

显示以上界面表示连接成功，已经可以使用树莓派的命令行功能了。
